# React-TypeScript

Minimal boilerplate for a single-page app using React, TypeScript 1.6 with TSX, and Visual Studio Code.

Usage:

* Install Node.js
* `npm install -g tsd`
* `npm install`
* `tsd install`
* `npm run build`

To view the app:

* `npm install -g http-server`
* `http-server`
* Open [http://localhost:8080/](http://localhost:8080/) in your browser of choice.
